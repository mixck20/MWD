How to cut audio & video files

  • Online Application ( for normal users )

    1. Audio Cutter
    2. Video Cutter

  • FFmpeg ( for users who can use shell of Window/Linux/MacOS )

    1. Install FFmpeg or download static builds.
    2. Run command:
      ffmpeg -y -i [input] -ac 1 -ar 8000 -ss [offset] -t [duration] out.wav
      Example:
      Cut 20 seconds of input.mp4 from 00:00:30 and save it to out.wav
      ffmpeg -y -i input.mp4 -ac 1 -ar 8000 -ss 00:00:30 -t 00:00:20 out.wav
